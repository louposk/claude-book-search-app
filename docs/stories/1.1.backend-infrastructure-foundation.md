### Database Schema Design
**Books Table** [Source: Epic requirements + current Book model]:
```sql
books (
  id TEXT PRIMARY KEY,           -- Google Books API ID
  title TEXT NOT NULL,
  authors TEXT,                  -- JSON array as string
  description TEXT,
  published_date TEXT,
  publisher TEXT,
  page_count INTEGER,
  categories TEXT,               -- JSON array as string
  image_links TEXT,             -- JSON object as string
  average_rating REAL,
  ratings_count INTEGER,
  language TEXT,
  preview_link TEXT,
  info_link TEXT,
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP
)
```

**Favorites Table** [Source: Epic requirements]:
```sql
favorites (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  book_id TEXT NOT NULL,         -- References books.id
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (book_id) REFERENCES books(id)
)
```

**Reading Lists Table** [Source: Epic requirements]:
```sql
reading_lists (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  name TEXT NOT NULL,
  description TEXT,
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP
)

reading_list_books (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  list_id INTEGER NOT NULL,
  book_id TEXT NOT NULL,
  added_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (list_id) REFERENCES reading_lists(id),
  FOREIGN KEY (book_id) REFERENCES books(id)
)
```

**Search History Table** [Source: Epic requirements]:
```sql
search_history (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  query TEXT NOT NULL,
  search_type TEXT NOT NULL,     -- 'title', 'author', 'isbn', 'general'
  results_count INTEGER,
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP
)
```

### API Endpoints Specification
**Books API** [Source: Epic requirements]:
- GET /api/books - List all saved books
- POST /api/books - Save a book from search results
- GET /api/books/:id - Get specific book details
- DELETE /api/books/:id - Remove book from library

**Favorites API** [Source: Epic requirements]:
- GET /api/favorites - List all favorite books
- POST /api/favorites - Add book to favorites
- DELETE /api/favorites/:bookId - Remove from favorites

**Reading Lists API** [Source: Epic requirements]:
- GET /api/reading-lists - List all reading lists
- POST /api/reading-lists - Create new reading list
- PUT /api/reading-lists/:id - Update reading list
- DELETE /api/reading-lists/:id - Delete reading list
- POST /api/reading-lists/:id/books - Add book to list
- DELETE /api/reading-lists/:id/books/:bookId - Remove book from list

**Search History API** [Source: Epic requirements]:
- GET /api/search-history - Get recent search queries
- POST /api/search-history - Record new search query
- DELETE /api/search-history/:id - Remove search entry
